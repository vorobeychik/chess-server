# EVO CHESS-SERVER

This project was created to evolution ts-bootcamp 2022

## Tech

 - TypeScript
 - node js
 - express
 - MongoDB
 - EsLint
 - Pritier
 - Jest
 


## Installation

You can clone this repository and run client app locally on your machine.

Then open the terminal and run these commands.

```
npm install
npm server:watch
```
